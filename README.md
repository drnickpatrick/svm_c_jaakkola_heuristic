# svm_c_jaakkola_heuristic
Compute Jaakkola's heuristic for sigma parameter of RBF-kernel SVM classifier using Shark library

This is a small utility to compute Jaakkola heuristic 
for G parameter of RBF-kernel SVM classifier using Shark library (http://image.diku.dk/shark/).
